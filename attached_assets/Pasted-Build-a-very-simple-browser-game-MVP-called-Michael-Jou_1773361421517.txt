Build a very simple browser game MVP called "Michael Journey" using Phaser.js.

The game should be mobile-first but also work on desktop browser.

Game concept:
- Cute, funny, slightly chaotic arcade-style endless vertical platform jumper
- The player controls a full-body pixel-art character named Michael
- The goal is to jump upward across platforms and survive as long as possible
- The score should increase by 1 for each new platform successfully landed on
- The game ends when the player falls below the screen

Core gameplay requirements:
- Start screen with the title "Michael Journey" and a Start button
- Endless upward-scrolling gameplay
- Randomly generated platforms
- Manual jump mechanic
- Left and right movement
- Score counter during gameplay
- Game over screen with final score and Restart button
- Basic jump sound effect
- Responsive layout for both phone and desktop

Controls:
Mobile:
- Hold left side button/zone to move left
- Hold right side button/zone to move right
- Tap jump button to jump

Desktop:
- A/D or Left/Right arrows to move
- Spacebar or Up arrow to jump

Visual style:
- Pixel-art look
- Cute and simple
- Funny and arcade/high-score focused
- Keep the UI clean and readable on phone screens
- Portrait-friendly for mobile, but should still work on desktop

MVP scope only:

- No collectibles
